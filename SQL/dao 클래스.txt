DAO 클래스
  ㄴ Data Access Object (데이터 접근 객체): 
	소프트웨어 개발에서 사용되는 디자인 패턴으로, 
	데이터베이스나 다른 데이터 소스에 접근하는 방법을 정의하는 객체입니다. 
	이 패턴은 데이터베이스와 애플리케이션의 로직을 분리하여 유지보수성을 높여줍니다.
  ㄴ SQL 실행을 위한 메소드로만 구성이 되는 클래스 정의
  ㄴ 일반적으로 고객, 상품, 구매 테이블에 해당하는 클래스를 각각 정의
  ㄴ Connection 객체 만드는 코드도 포함.

SQL명령어	매개변수				결과값

Insert		컬럼데이터(들) VO클래스		int
update		컬럼데이터(들) VO클래스		int
delect		1개 컬럼값이 대부분.(특정기능제외)	int	

select		조회 값(들)(0포함 n개)		컬럼데이터(들) VO클래스
