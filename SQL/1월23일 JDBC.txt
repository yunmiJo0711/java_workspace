1. 외부 라이브러리
- 특정 기능 제공을 위해 개발되어 배포되는 클래스들의 집합.
- 별도의 추가 작어이 필요(자바 프로젝트에 외부라이브러리 추가필요)
	+ vs code에서는 Referenced Libraries 항목 찾아서 + 클릭해서 jar 추가
-jar파일로 배포되는 것은 클래스들의 압축파일.
-예시 : 	gson.jar -> 자바객체와 json 문자열 사이의 변환
	lombok.jar -> 어노테이션 기반으로 자주 사용하는 코드 생성
	ojdbc8.jar -> 다른 프로그램에서 오라클과 연결되는 드라이버 

-JAR 파일을 어디서 구할까요?
	저장소 사이트  또는 각 프로젝트 사이트 에서 다운로드 
	* 롬복은 해당 사이트 검색해서 다운로드합니다. 
	* 오라클 드라이버는 오라클 설치된 상태이므로 설치폴더에서 복사해옵니다. 
	-> C:\app\Class01\product\18.0.0\dbhomeXE\jdbc\lib
